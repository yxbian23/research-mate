# Academic Integrity Rule

学术诚信规范，确保 AI 辅助学习的合理使用。

---

## 核心原则

### 1. 不编造原则 (No Fabrication)

**定义**: 所有生成的内容必须基于提供的材料，不编造不存在的内容。

**必须做到**:
- 引用的定理、公式必须在讲义中存在
- 解题方法必须基于课程教授的内容
- 不确定的内容必须明确标注

**示例**:
```
✓ 正确: "根据 Lecture 3, Page 15 的 Theorem 2.1..."
✓ 正确: "这个结论在提供的材料中未找到，建议查阅教材..."
✓ 正确: "[不确定] 这一步可能需要额外验证..."

✗ 错误: 编造一个讲义中不存在的定理
✗ 错误: 使用课程未教授的高级方法而不说明
✗ 错误: 假装某内容在讲义中而实际并没有
```

### 2. 辅助而非代替 (Assist, Not Replace)

**定义**: AI 工具是学习辅助，不是作业代做。

**必须做到**:
- 帮助理解，而非提供直接答案
- 鼓励学生独立思考
- 所有输出需要学生验证和修改

**期望的使用方式**:
```
1. 学生先尝试自己解题
2. 遇到困难时使用 AI 辅助理解
3. 理解后用自己的方式重新写
4. 最终提交自己的作品
```

### 3. 来源标注原则 (Source Attribution)

**定义**: 所有内容必须标注来源。

**标注格式**:
```markdown
**定理来源**: Theorem 2.1 (Lecture 3, Page 15)
**公式来源**: Equation (5), Lecture 4, Page 23
**方法来源**: Example 4.2, TA Session 3
**不确定**: [需要验证] 这一结论可能需要额外确认
```

### 4. 不确定标注原则 (Uncertainty Marking)

**定义**: 对不确定的内容必须明确标注。

**何时标注**:
- 讲义中未明确提到的推导
- 可能存在多种解法的情况
- 边界情况或特殊条件
- 对结果正确性不完全确定时

**标注方式**:
```markdown
[不确定] 这一步的推导可能有其他方法...
[需验证] 请检查这个计算是否正确...
[可能] 这个结论可能需要额外条件...
[建议] 建议查阅教材第X章确认...
```

## 不同场景的诚信要求

### 讲义总结

| 允许 | 禁止 |
|------|------|
| 提炼和组织讲义内容 | 添加讲义中没有的内容 |
| 用更清晰的方式解释概念 | 编造示例而不标注 |
| 标注来源和页码 | 省略来源标注 |

### 作业辅助

| 允许 | 禁止 |
|------|------|
| 解释解题思路 | 直接提供可提交的答案 |
| 展示详细步骤供学习 | 鼓励直接抄袭 |
| 指出相关讲义内容 | 使用课程未教的方法而不说明 |
| 交叉验证结果 | 保证答案100%正确而不验证 |

### 复习材料

| 允许 | 禁止 |
|------|------|
| 整理和组织课程内容 | 添加超出课程范围的内容 |
| 创建练习题（标注来源/自编） | 伪造历年考题 |
| 总结易错点 | 保证覆盖考试内容 |

## 学生责任

### 使用前

- [ ] 了解课程的学术诚信政策
- [ ] 明确 AI 辅助工具的使用限制
- [ ] 先尝试自己解决问题

### 使用中

- [ ] 将 AI 输出视为参考而非答案
- [ ] 验证所有关键步骤和结果
- [ ] 标记需要进一步确认的内容

### 使用后

- [ ] 用自己的语言重新表述
- [ ] 确保理解每一步推导
- [ ] 独立完成最终提交的作业

---

## AI 辅助工具的责任

### 必须做到

1. **提醒学术诚信**: 在提供作业帮助时提醒学生诚信使用
2. **标注来源**: 所有内容标注来源
3. **标注不确定**: 不确定的内容明确标注
4. **鼓励独立**: 鼓励学生独立思考和验证

### 禁止做到

1. **不保证正确**: 不声称答案一定正确
2. **不鼓励抄袭**: 不鼓励直接提交
3. **不编造内容**: 不编造材料中没有的内容
4. **不用于考试**: 不在考试场景中使用

---

## 违规后果说明

**注意**: 以下是一般性的学术诚信违规后果说明，具体后果由学校和课程决定。

### 常见后果

1. **作业零分**: 该次作业得零分
2. **课程不及格**: 严重情况下课程成绩为 F
3. **学术处分**: 记入学术诚信档案
4. **严重后果**: 可能导致开除学籍

### 预防措施

1. 始终将 AI 输出作为学习参考
2. 用自己的方式重新写作业
3. 在不确定时咨询老师或 TA
4. 保留学习和理解的过程记录

## 检查清单

在提交作业前确认：

- [ ] 我理解了所有解题步骤
- [ ] 我能够独立解释每一步推导
- [ ] 我用自己的语言重新写了答案
- [ ] 我验证了关键结果的正确性
- [ ] 我的提交符合课程学术诚信政策
- [ ] 如果使用了 AI 辅助，我遵循了使用规范

## 正确使用示例

### 示例：正确使用 AI 辅助学习

```
场景: 不理解 Lecture 3 中的纳什均衡定义

正确做法:
1. 先阅读讲义原文
2. 使用 AI 解释概念
3. 阅读 AI 提供的示例
4. 自己用不同例子验证理解
5. 用自己的话写学习笔记

错误做法:
1. 直接让 AI 总结
2. 复制 AI 的总结作为笔记
```

### 示例：正确使用作业辅助

```
场景: 作业题要求证明某定理

正确做法:
1. 先尝试自己证明
2. 卡住时查看 AI 提供的思路
3. 理解思路后关闭 AI
4. 自己重新写证明
5. 对比检查遗漏

错误做法:
1. 直接让 AI 写证明
2. 复制 AI 的证明提交
```
